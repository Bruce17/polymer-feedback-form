Polymer Feeback Form Demo Application

Please click on the "feedback" button on the left to test this application.

Options

Provide options:

url Backend url to which the form content will be sent (required).
route Called route (current site) e.g. a Symfony route "site_index_index".
request-uri Raw request uri without host e.g. "blog/some/page.html"
name Prefilled username e.g. if user is alreay logged in.
email Prefilled email e.g. if user is already logged in.

Example

<feedback-element url="localhost"></feedback-element>
